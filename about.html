About TaskFlow

TaskFlow is a lightweight task management web application designed to help you organize and track your tasks efficiently.

How to Use

1. Go to Home to see your task summary.

2. Click Add Task to create a new task.

3. View all tasks in Task List and filter by status.

4. Click any task to see its Task Detail and manage it.
